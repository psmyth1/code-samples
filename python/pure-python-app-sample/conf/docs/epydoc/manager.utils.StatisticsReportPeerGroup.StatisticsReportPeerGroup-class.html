Please refer to this note in "StatisticsReport"

The same overall structure is used in this class as in "StatisticsReport."  I decided to create
it as a separate module due to minor structural differences, but the overall theme is the same.

Structural Differences
- only generating one report sheet with the specified peer group funds and possibly a MCPFAI sheet
- not using a specified invested, focus, or both fund universe

Consequently, changes had to be made to the data exporting loops, etc.  Instead of adding logic
to handle this in "StatisticsReport," which would muddle up the logic of that code, I decided it
would be easier to just create a new class.
